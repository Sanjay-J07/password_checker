# 🔐 Password Strength Checker

## 📌 Description

A Python-based tool that evaluates the strength of a password using common security rules. It analyzes user input and provides feedback on whether the password is weak, medium, or strong, along with suggestions for improvement.

---

## 🚀 Features

* 🔍 Checks password strength (Weak / Medium / Strong)
* 📊 Uses scoring system for accurate evaluation
* 🧠 Validates:

  * Minimum length
  * Lowercase letters
  * Uppercase letters
  * Numbers
  * Special characters
* 💡 Provides improvement suggestions
* 🔁 Allows multiple checks (loop-based program)

---

## 🧠 How It Works

1. User enters a password
2. Program checks different conditions using regex
3. Assigns a score based on criteria
4. Displays strength level
5. Suggests improvements if needed

---

## 🛠️ Technologies Used

* Python
* Regular Expressions (`re` module)

---

## ▶️ How to Run

```bash
python password_checker.py
```
